This project is for automated medical review. 

It is recommended to download the data directly from the bottom link of the project 
because the data size of the project is large.

This data consists of categorical data and character data.
Thus, the use of classification algorithms and simple natural language processing 
to extract data favorable to automated classification 
contributes to the automation of medical review faster.
